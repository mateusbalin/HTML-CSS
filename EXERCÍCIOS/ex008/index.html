Texto Marcado

Podemos usar um texto marcado usando a tag MARK.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido (sublinhado) e dar uma ênfase ou indicar que ele foi adicionado depois. Existe tambêm a tag U porém ela não é semântica.

Texto sobrescrito e subscrito

Para que o texto fique sobrescrito, a tag utilizada é sup. Exemplo: 28 Já para que fique subscrito, a tag utilizada é sub. Exemplo: H2O
